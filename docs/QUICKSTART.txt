üåê VIEW ON YOUR PHONE

Option 1: Using Local Network (Best)
-------------------------------------
1. Find your computer's IP address:
   - Windows: ipconfig (look for IPv4 Address)
   - Mac: System Preferences > Network

2. Put the project folder on your desktop

3. Open terminal/command prompt in the project folder

4. Run: python -m http.server 8000

5. On your phone, open: http://YOUR_IP:8000
   (Replace YOUR_IP with your computer's IP)

Option 2: Using Local File
---------------------------
Just open index.html from your phone's file browser

üÜò TROUBLESHOOTING

- Check file paths are correct

- Verify file names match (case sensitive)
- Check HTML syntax

WhatsApp not opening:
- Make sure WhatsApp is installed
- Check phone number format is correct

Contact form not working:
- Check JavaScript is enabled
- Ensure all form fields are filled

Styling looks wrong:
- Clear browser cache (Ctrl+Shift+Del)
- Check css/style.css is loaded
- Try different browser
